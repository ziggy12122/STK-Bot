2. Additional Files

1. `keep_alive.py` (to prevent Replit from sleeping):
   ```python
   from flask import Flask
   from threading import Thread
   
   app = Flask('')
   
   @app.route('/')
   def home():
       return "STK Shop Bot Alive"
   
   def run():
       app.run(host='0.0.0.0', port=8080)
   
   def keep_alive():
       t = Thread(target=run)
       t.start()
   ```
2. `.env` (for secrets):
   ```
   DISCORD_TOKEN=your_bot_token_here
   ```
